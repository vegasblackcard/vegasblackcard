## RC Config - Remote Access Manager

A lightweight shell-based tool for managing SSH remote connections, tunnels, and file syncing.

### Quick Start

```bash
source rc-config.sh
rc hosts            # List configured hosts
rc connect dev      # SSH into a host
rc tunnel db dev    # Open a port-forwarding tunnel
rc sync staging ./app /opt/app  # Rsync files
```

### Configuration

Edit `.remoterc` to define your hosts, tunnels, and SSH settings. See the file for all available options.
